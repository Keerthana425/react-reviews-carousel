
# React Reviews Carousel

A simple and interactive Reviews Carousel built using React. Users can browse customer reviews using left and right navigation buttons.

## Features

- Display customer reviews
- Navigate between reviews
- React Class Components
- State Management using `setState`
- Responsive UI

## Technologies Used

- React JS
- JavaScript
- CSS3
- HTML5

## Project Preview

- User image
- Username
- Company name
- Review description
- Previous and Next navigation buttons

## Installation

1. Clone the repository

```bash
git clone <your-repository-link>
```

2. Navigate to project folder

```bash
cd react-reviews-carousel
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```
